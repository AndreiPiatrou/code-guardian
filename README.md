# code-guardian
Simple cli tool to check a code against different kinds of issues
### Currently supported checks

- sensitive data: keys, secrets, passwords

### How to install

```bash
npm i -g code-guardian
```

### How to start

```bash
cd your-repo
code-guardian

# for more details
code-guardian --help
```
